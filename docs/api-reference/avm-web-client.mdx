## Static Methods

### `init([options])`

> Initializes the AVM web client object. If no options are supplied, a default configuration is used.

#### Parameters

| Name    | Type                                                         | Required | Default | Description                                                       |
|---------|--------------------------------------------------------------|----------|---------|-------------------------------------------------------------------|
| options | [`IAVMWebClientInitOptions`](#iavmwebclientinitoptions)  | no       | -       | An object to change the initial configuration.                    |

#### Returns

| Type                                | Description                             |
|-------------------------------------|-----------------------------------------|
| [`AVMWebClient`](#avmwebclient) | An initialized AVM web client object. |

## Methods

### `getConfig()`

> Returns the configuration.

#### Returns

| Type                                | Description                             |
|-------------------------------------|-----------------------------------------|
| [`IAVMWebClientConfig`](#iavmwebclientconfig) | The client's configuration. |

## Types

### `IAVMWebClientConfig`

| Name        | Type      | Required | Default | Description                                                      |
|------------|-----------|----------|---------|--------------------------------------------------------------|
| debug | `boolean` | no       | `false` | Outputs debug information to the console.                 |
