## Installation

### Using a package manager

Use your favourite package manager to install:

```bash npm2yarn
npm install --save @agoralabs-sh/avm-web-provider
```

### Download via a CDN

Include the minified browser bundle directly in your HTML like so:

```html
<html>
  ...
  <script src="https://github.com/agoralabs-sh/avm-web-provider/releases/latest/download/avm-web-client.min.js"></script>
  ...
</html>
```
