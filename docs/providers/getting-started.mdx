## Installation

Use your favourite package manager to install:

```bash npm2yarn
npm install --save @agoralabs-sh/avm-web-provider
```

## Initialization

Firstly, we want to initialize the AVM provider with an ID:

<Tabs
  defaultValue="javascript"
  values={[
    { label: 'Javascript', value: 'javascript' },
    { label: 'TypeScript', value: 'typescript' },
  ]}>
  <TabItem value="javascript">

    ```js
    const { AVMWebProvider } = require('@agoralabs-sh/avm-web-provider');

    const provider = AVMWebProvider.init('02657eaf-be17-4efc-b0a4-19d654b2448e');
    ```

  </TabItem>
  <TabItem value="typescript">

    ```typescript
    import { AVMWebProvider } from '@agoralabs-sh/avm-web-provider';

    const provider: AVMWebProvider = AVMWebProvider.init('02657eaf-be17-4efc-b0a4-19d654b2448e');
    ```

  </TabItem>
</Tabs>

:::note

The provider ID is what will identify your provider with clients. It is recommended that you use a [UUIDv4](https://www.rfc-editor.org/rfc/rfc4122) compliant string.

:::

:::note

See [`IAVMWebProviderInitOptions`](../../api-reference/types#iavmwebproviderinitoptions) for more information on what options are supported.

:::
