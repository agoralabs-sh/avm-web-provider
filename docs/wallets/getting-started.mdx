## Installation

You can use your favourite package manager to install:

```bash npm2yarn
npm install --save @agoralabs-sh/avm-web-provider
```

## Initialization

Firstly, we want to initialize the wallet object with a provider ID:

<Tabs
  defaultValue="javascript"
  values={[
    { label: 'Javascript', value: 'javascript' },
    { label: 'TypeScript', value: 'typescript' },
  ]}>
  <TabItem value="javascript">

    ```js
    const { AVMWebWallet } = require('@agoralabs-sh/avm-web-provider');

    const provider = AVMWebWallet.init('02657eaf-be17-4efc-b0a4-19d654b2448e');
    ```

  </TabItem>
  <TabItem value="typescript">

    ```typescript
    import { AVMWebWallet } from '@agoralabs-sh/avm-web-provider';

    const provider: AVMWebWallet = AVMWebWallet.init('02657eaf-be17-4efc-b0a4-19d654b2448e');
    ```

  </TabItem>
</Tabs>

:::note

The provider ID is what will identify your wallet with dApps. It is recommended that you use a [UUIDv4](https://www.rfc-editor.org/rfc/rfc4122) compliant string.

:::

:::note

See [`IAVMWebWalletInitOptions`](../../api-reference/avm-web-wallet#iavmwebwalletinitoptions) for more information on what options are supported.

:::
